A terminal centric distro written from scratch

> Features:-

  • nemesis-pkg: A package manager supporting basic operations. Written from scratch
  • Extremely stable and optimised packages
  • Works on any x86_64 based processor
  • runit is the default init system and services are patched to work with runit

> System Requirements:-

  • CPU:- Any x86_64 based processor having SSE2(Stremaing SIMD Extensions 2)
  • RAM:- 512MB(No Firefox), 1.5GiB(Working browser)
  • Disk Space:- idk?
